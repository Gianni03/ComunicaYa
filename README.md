ComunicaYa is a personal project to build a low-cost Augmentative and Alternative Communication (AAC) tool, aimed at simplifying daily communication for children with Autism Spectrum Disorder (ASD) and limited functional language. The app focuses on basic requests, functional refusal and routine visualization.

### ‚öôÔ∏è Key features
- Accessibility-centered design: high contrast, large typography, primary colors.
- Touch-optimized for tablets and mobiles.
- TTS (Text-to-Speech): each button plays a clear phrase when pressed (Web Speech API).
- High-impact buttons with strong visual feedback.
- Simple thematic organization to reduce sensory overload:
  - Quick Wishes: "I want" / "I don't want" and immediate requests.
  - My Routine: pictograms for daily activities (Eat, School, Bathroom).
  - My Needs: internal and emotional states (Happy, Sad, Thirsty, Hurt).
